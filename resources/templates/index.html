PHP ユーザーにとって、PEAR ライブラリはとても便利なものですが、そのライブラリを自分で作って公開することはとても気軽にできることではありませんでした。 openpear を使うと、用意された Subversion のリポジトリから自動でパッケージングし openpear のチャンネルへの登録も同時に行うことが出来ます。PHP 開発者はこの機会に是非登録して、あなたの作成したライブラリを共有しませんか？

How to use?

openpear に登録されているパッケージは、すべて本家 PEAR に登録されているパッケージと同様に使用することができます。PEAR コマンドが使用できる環境で以下のコマンドを入力してください。

pear channel-discover {$rhaco.constant('CHANNEL')}

パッケージのインストールは次のようにできます。

pear install openpear/パッケージ名

各パッケージの詳細な使用方法はパッケージ毎のページをご覧下さい。
